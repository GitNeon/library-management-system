# 一、项目介绍
C++练习项目，图书管理系统(library-management-system)

# 二、目录说明

```text
library-management-system/
├── .gitignore               # Git 忽略文件（你已经有了）
├── README.md                # 项目说明（必备）
├── CMakeLists.txt           # 构建工具（企业标准）
├── src/                     # 源代码（核心业务）
│   ├── core/                # 核心类：Book、User、Library
│   ├── utils/               # 工具函数：字符串、时间、校验
│   └── main.cpp             # 程序入口
├── include/                 # 对外头文件（大型项目必备）
├── bin/                     # 编译后的可执行文件
├── build/                   # CMake 编译目录
├── docs/                    # 文档（设计文档、使用说明）
├── data/                    # 图书/用户数据文件（.txt / .dat）
└── test/                    # 单元测试（工程化必备）
```
