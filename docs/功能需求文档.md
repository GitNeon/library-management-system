# 图书管理系统（C++）功能需求文档
## 一、文档概述
本文档用于明确**C++ 控制台版图书管理系统**的全部功能需求、模块划分、操作逻辑，作为代码开发的直接依据。系统面向**图书管理员**使用，无普通用户权限，基于控制台交互实现图书信息管理、借阅/归还、查询统计等核心功能。

## 二、系统整体定位
1. **运行环境**：Windows/Linux 控制台，纯 C++ 代码实现，无图形界面
2. **数据存储**：基础版使用内存存储（程序关闭数据清空），进阶版支持文件持久化存储
3. **使用角色**：仅管理员（包含登录验证功能）
4. **核心目标**：简洁、易用、完整覆盖图书管理基础业务

## 三、核心功能模块划分（共 6 大模块）
### 模块1：管理员登录模块（系统入口）
1. 启动程序后，首先进入管理员登录界面
2. 预设管理员账号/密码（可自定义）
3. 验证失败：提示错误，允许重新输入（可限制次数）
4. 验证成功：进入主功能菜单
5. 支持退出系统功能

### 模块2：图书信息管理模块（核心基础）
1. **添加图书**
   - 录入图书信息：图书编号、书名、作者、出版社、出版日期、总数量、当前可借数量
   - 校验：图书编号不可重复
2. **删除图书**
   - 按**图书编号**精准删除图书
   - 校验：若图书存在未归还记录，禁止删除（保证数据安全）
3. **修改图书信息**
   - 按图书编号查找图书
   - 可修改：书名、作者、出版社、数量等信息（不可修改图书编号）
4. **查看所有图书**
   - 以表格形式控制台输出全部图书列表
   - 显示：编号、书名、作者、可借数量、总数量、状态（可借/已借完）

### 模块3：图书查询模块
1. **精准查询**：按图书编号查询，直接显示单本图书完整信息
2. **模糊查询**：按书名/作者关键词查询，输出匹配的所有图书列表
3. 查询无结果时，给出友好提示

### 模块4：图书借阅与归还模块
1. **图书借阅**
   - 输入：图书编号、借阅人姓名、借阅日期
   - 校验：图书是否存在、可借数量是否大于0
   - 借阅成功：可借数量 -1，记录借阅信息
2. **图书归还**
   - 输入：图书编号、借阅人姓名
   - 校验：是否存在对应借阅记录
   - 归还成功：可借数量 +1，更新借阅状态
3. **查看借阅记录**
   - 查看所有未归还的借阅记录
   - 显示：图书编号、书名、借阅人、借阅日期

### 模块5：数据统计模块
1. 统计图书馆总图书数量（总册数）
2. 统计可借阅图书数量
3. 统计已借出图书数量
4. 统计图书馆图书种类数

### 模块6：系统工具模块
1. **菜单交互**：清晰的控制台菜单，输入数字选择功能
2. **数据持久化（进阶）**
   - 程序关闭时：将图书信息、借阅记录保存到本地文件（.txt）
   - 程序启动时：从文件加载数据，避免数据丢失
3. **清空屏幕**：优化控制台显示效果
4. **友好提示**：操作成功/失败、输入错误、格式错误提示

## 四、数据结构设计（对应代码）
### 1. 图书结构体（Book）
```cpp
struct Book {
    string id;        // 图书编号（唯一）
    string name;      // 书名
    string author;    // 作者
    string publisher; // 出版社
    string date;      // 出版日期
    int total;        // 总数量
    int current;      // 可借数量
};
```

### 2. 借阅记录结构体（Borrow）
```cpp
struct Borrow {
    string bookId;    // 借阅的图书编号
    string bookName; // 图书名
    string userName; // 借阅人姓名
    string borrowDate; // 借阅日期
};
```

## 五、系统操作流程
1. 启动程序 → 管理员登录
2. 登录成功 → 显示主菜单
3. 主菜单选择功能：
   - 图书管理（增/删/改/查所有）
   - 图书查询（精准/模糊）
   - 借阅/归还
   - 数据统计
   - 保存/退出
4. 执行功能 → 返回菜单 → 循环操作
5. 选择退出 → （可选）保存数据 → 关闭程序

## 六、基础版 / 进阶版功能区分
### 基础版（必做）
- 内存存储数据
- 管理员登录
- 图书增删改查
- 借阅归还
- 基础统计
- 控制台菜单交互

### 进阶版（可选扩展）
- 文件存储（数据持久化）
- 借阅逾期提醒
- 多管理员权限
- 按可借状态筛选图书
- 图书排序（按编号/数量）
