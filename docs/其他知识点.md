# 一、C++ 中的常见命名前缀

| 前缀 | 全称 | 含义 | 示例 |
|------|------|------|------|
| m_ | Member | 类的成员变量（非静态） | m_id, m_name |
| s_ | Static | 类的静态成员变量 | s_instance, s_count |
| g_ | Global | 全局变量 | g_config, g_logger |
| c_ | Constant | 常量（有时用 k_） | c_max_size, kPi |
| 无前缀 | - | 局部变量、函数参数、普通函数名 | id, count, calculate() |
